# MeuVerso 🖋️

**MeuVerso** é o seu refúgio poético. Um espaço dedicado à escrita, leitura e compartilhamento de versos que tocam a alma. Explore temas variados, siga seus autores favoritos e publique suas próprias criações em uma interface elegante e minimalista. Transforme sentimentos em palavras com o MeuVerso.

---

## ✨ Funcionalidades

- ✍️ **Criação de Versos:** Ferramenta simples e elegante para escrever seus poemas.
- 📖 **Leitura Imersiva:** Design focado na legibilidade para uma experiência de leitura tranquila.
- 👥 **Comunidade:** Siga seus autores favoritos e descubra novos talentos.
- 🎨 **Interface Minimalista:** Foco total no conteúdo, sem distrações.
- 📱 **Mobile First:** Desenvolvido nativamente para Android.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias modernas do ecossistema Android:

- **Linguagem:** Java/Kotlin (Android Nativo)
- **UI:** XML Layouts / Material Design 3
- **SDK Mínimo:** 21 (Android 5.0)
- **Target SDK:** 35 (Android 15)
- **Gerenciamento de Build:** Gradle
- **Automação:** Fastlane (Metadata & Store Management)

## 🚀 Como Compilar o Projeto

Para rodar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/MeuVerso.git
   ```
2. Abra o projeto no **Android Studio**.
3. Aguarde a sincronização do Gradle.
4. Conecte um dispositivo físico ou inicie um emulador.
5. Clique em **Run** (ícone de play).
